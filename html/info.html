快速开始

  1. 固定扩展图标：点击浏览器工具栏右侧的拼图图标，找到 Cloaq 并点击图钉图标将其固定到工具栏。
  2. 点击工具栏上的 Cloaq 图标打开设置面板。
  3. 选择配置方案或自定义设置时区、语言和地理位置。
⚡

Cloaq 工作原理

Cloaq 使用 Chrome Debugger API 直接在浏览器层面修改数据，这使得它能够在所有 iframe 和 Web Worker 中生效。与其他依赖脚本注入的扩展不同，Cloaq 的方法更加可靠，确保数据修改无法被检测，并且在页面初始加载时就能正常工作。

关于 IP 地址

⚠️
重要提示：Cloaq 不会更改您的 IP 地址。如需更改 IP 地址，请使用 VPN 或代理服务。

Cloaq 的作用是同步您的浏览器位置数据与 VPN 的 IP 地址位置，使网站无法通过时区和地理位置检测发现您使用了 VPN。

隐藏调试通知栏

当扩展运行时，浏览器顶部会显示一个调试通知栏。如需隐藏此通知栏，可以使用 --silent-debugger-extension-api 启动参数运行 Chrome。
